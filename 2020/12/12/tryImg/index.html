  1. 语言背景

  2. 程序逻辑

    • 编程语言环境
      • 强类型：
        • 定义a = “string”，就知道a是一个字符串类型
        • Java，python，js
      • 弱类型：
        • 需要定义 String a = “String”，才知道类型，像C需要程序猿去判定类型
        • C，PHP

  3. 变量

    • 变量定义
      • 动态类型
      • 静态类型
    • 变量类型
      • 值类型（基本类型）
        • 变量在栈里面声明，
        • C，C++，引用类型通过指针来实现
      • 引用类型（封装类型）
        • 变量是指针，指向了内存地址中变量的地址
        • python，js
      • java - 即保留值类型，也保留引用类型
      • 应用在比较变量中，引用类型需要重写equal方法，不然比较的是地址

  4. 容器对象

    • 简单操作
    • 浅复制，深复制、序列化
